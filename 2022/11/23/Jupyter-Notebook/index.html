Jupyter Notebook的常见操作-需要熟记！

Jupyter Notebook常见快捷键操作

首先要知道Jupyter有三种cell类型

  1. Code
  2. Markdown
  3. Raw NBConvert（普通文本，运行不会输出结果）

两种模式

  1. 编辑模式（Enter）
  2. 命令模式（Esc）

1.两种模式都可以使用的快捷键

  • Shift+Enter执行本单元代码，并且跳转到下一单元
  • Ctrl+Enter执行本单元代码，留在本单元

2.命令模式

  • Y：cell切换到Code模式
  • M：cell切换到Markdown模式
  • A：在当前cell的上面添加cell
  • B：在当前cell的下面添加cell
  • 双击D：删除当前cell
  • Z：回退
  • Ctrl+Shift+减号：分隔cell，在光标处
  • L：为当前cell加上行号

3.编辑模式：按Enter或鼠标单击代码块内部进入

  • **Ctrl+鼠标单击(Mac:CMD+鼠标单击)**：多光标操作
  • **Ctrl+Z(Mac:CMD+Z)**：回退
  • Tab键：代码补全
  • **Ctrl(Mac:CMD+/)**：注释多行代码
